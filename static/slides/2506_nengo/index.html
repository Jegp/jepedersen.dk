NIR's Design Philosophy

  • Continuous-time foundation: Hybrid systems combining continuous-time dynamics and discrete events
  • Platform agnostic: Abstracts hardware constraints while preserving computational essence
  • Future-proof: Compatible with today's tools, ready for tomorrow's hardware

Demo 1: Build Once, Run Anywhere

# Train in snnTorch
import norse
import snntorch
import sinabs

# Create and train model
model = snntorch.Leaky(beta=0.9, init_hidden=True)
# ... training code ...

# Export to NIR (1 line!)
nir_model = snntorch.export_to_nir(model)

# Deploy to Norse (1 line!)
norse_model = norse.torch.from_nir(nir_model)

# Deploy to hardware (1 line!)
sinabs_model = sinabs.from_nir(nir_model)

Result: Same model, multiple platforms, minimal effort!

The Reproducibility Crisis in Neuromorphics

  • Platform-specific implementations create validation bottlenecks
  • Hidden assumptions in discretization and hardware constraints
  • Version dependencies make long-term reproducibility challenging

NIR as a Reproducibility Tool

  • Canonical representation: Mathematical ground truth independent of implementation
  • Version control for models: NIR graphs as portable scientific artifacts
  • Cross-platform validation: Same model, multiple implementations, verifiable results

Demo 3: NIR Primitives Under the Hood

# Create a LIF neuron primitive
import nir
import numpy

lif = nir.LIF(
    tau=numpy.array([20.0]),         # Time constant (ms)
    v_threshold=numpy.array([1.0]),  # Spike threshold
    v_reset=numpy.array([0.0]),      # Reset potential
    r=numpy.array([1.0])             # Resistance
)
synapse = nir.Linear(weight=numpy.array([0.5]))
graph = nir.NIRGraph(
    nodes={"lif": lif, "synapse": synapse},
    edges=[("lif", "synapse")],
)

Demo 4: Complete Workflow

# 1. Design in your favorite framework
import nengo
import nirviz

model = nengo.Network()
with model:
    input_node = nengo.Node([0])
    ensemble = nengo.Ensemble(100, 1)
    nengo.Connection(input_node, ensemble)

# 2. Export to NIR
nir_graph = nengo.export_nir(model)

# 3. Visualize the computational graph
nirviz.visualize(nir_graph)  # Shows graphical structure

# 4. Deploy everywhere
deploy_to_loihi(nir_graph)
deploy_to_spinnaker(nir_graph)
deploy_to_xylo(nir_graph)
