Any function that can be computed by an algorithm can be computed by a Turing machine. (Alonzo Church and Alan Turing, ~1936)

Church-Turing-Deutsch principle

Any physical process can be simulated by a universal computer to arbitrary precision. (David Deutsch, 1985)

Summary

  • Mathematical foundation: ODEs as computations physics is computation
  • Mixed-signal computing: Digital + analog
  • Performance advantage: OOM gains (sometimes)
  • Applications: Event-based sensing, real-time control, adaptive (antifragile) systems

  • Future outlook: Analog hardware scaling approaches fundamental limits; Plasticity and adaptability
